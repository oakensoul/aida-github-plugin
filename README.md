## Development

### Prerequisites

- Python 3.11+
- pip

### Setup

```bash
git clone https://github.com/YOUR_USER/aida-github-plugin.git
cd aida-github-plugin
pip install -e ".[dev]"
```

### Commands

```bash
make lint    # Run linters
make test    # Run tests
make clean   # Clean build artifacts
```
